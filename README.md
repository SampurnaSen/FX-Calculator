# FX Calculator
This project aims at creating a web-based currency converter application..

Input for To, From and Amount are taken from the UI inputs. Then based on the given data, the final exchane amount is displayed.

## Local Run
```
npm run start
```

Currency Converter

From: AUD
To: USD
Amount: 100

CLick on 'Conver'

Output: USD 26.96


## Run tests
```
npm run test
```
